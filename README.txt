A search function to query the directory of the Harvard Art Museums[1] by way of the Harvard Art Museums API[2]. 

The fonts are set on Authentic Sans[3] and Valkyrie[4].

1: https://harvardartmuseums.org/
2: https://github.com/harvardartmuseums/api-docs
3: https://authentic.website/sans.html
4: https://mbtype.com/fonts/valkyrie/
